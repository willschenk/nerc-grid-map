Dataset summary

How to read it

Each dot is one organization, at its headquarters.

Color role mix · Size role weight · Dotted no NERC ID

Bigger dot = more reliability roles (a grid authority); smaller dot = a single-role entity.
Color shows the mix of roles — see the abbreviations below.
Dotted outline = not in the NERC registry (no NERC ID).
